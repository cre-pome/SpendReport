# SpendReport
出費記録用アプリ

## 基本仕様
- タブは以下の2つ
	1. 今月の出費を確認する画面
	2. 月を選択してその月の出費状況を確認する画面≈

- 登録ボタンを押すと出費登録画面に移行する(登録ボタンはどのタブでも表示される)

- 出費登録画面では以下の項目を入力する
	1. 金額
	2. タグ(趣味、映画等)。タグは登録しなくても良い
	3. 出費時刻(デフォルトで現在時刻)
